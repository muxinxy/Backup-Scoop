# Scoop Backup Script

## Backup

Backup-Scoop.bat

修改为自适应路径，根据`scoop config root_path`的输出结果定义备份目录

## Restore

Restore-Scoop.bat

修改为自定义路径，根据用户输入定义恢复目录，默认D盘
